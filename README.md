# MyCampusMyHome

#### 介绍

我的校园我的家，校园宿舍管理系统，一个大学中的校园项目。

基础的增删改查不必说，还有登录拦击、记住密码等小功能，在学生时代实现了还是挺自豪的。

多角色：学生、楼宇管理员、系统管理员

## 软件架构

`SSM` + `MySQL`

## 安装教程

数据库与需求说明书在`data`文件夹中。

导入数据库后，IDEA打开，修改数据库配置文件即可。

用没用到Redis已经忘记了，应该没有吧。
